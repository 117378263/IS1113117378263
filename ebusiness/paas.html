Platform As A Service

Platform-as-a-service (PaaS) is a type of cloud computing offering in which a service provider delivers a platform to clients(such as cloud 9), enabling them to develop, run, and manage business applications without the need to build and maintain the infrastructure such software development processes typically require. Users typically access PaaS offerings via a web browser.

The Future Of Paas

The future of PaaS in cloud computing is evolving quickly. Technology is constantly growing exponentially while some PaaS vendors evolve linearly. Some PaaS solutions can lock you into a box, dictating languages and tools, one size fits all, while open source improves connection among servers and workers. Even from an expense standpoint, only the best PaaS providers include costs like deployment, infrastructure, management or support.
